# NIOM&CE – Nail Booking & Shop (FE demo)

Frontend demo cho nghiệp vụ NIOM&CE (B2C) với UI kiểu marketplace:

- Home: hero + thanh search + danh sách salon phổ biến
- Booking: chọn salon/dịch vụ/thợ/ngày/giờ trống và lưu booking vào localStorage
- Shop/Cart: mua sản phẩm, giỏ hàng, checkout giả lập (localStorage)

## Run

```bash
npm install
npm run dev
```

Mặc định chạy tại `http://localhost:5173/`.
